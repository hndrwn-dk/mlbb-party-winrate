# Public Folder

This folder contains static assets served by Next.js.

## Favicon

**Place your `favicon.ico` file here** (at `public/favicon.ico`).

The app is configured to automatically use it. See `FAVICON.md` in the root directory for detailed setup instructions.

## File Structure

```
public/
  favicon.ico          ‚Üê Place your favicon here
  [other static assets]
```

## Quick Setup

1. Create or download a favicon.ico file
2. Place it in this `public/` folder
3. The favicon will appear automatically in browser tabs

## Recommended

- Format: ICO (multi-resolution) or PNG
- Sizes: 16x16, 32x32, 48x48
- Theme: Match your gaming theme (green/teal colors)
